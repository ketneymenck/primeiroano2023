Agronomia e Pecuária

Perfil para aqueles que desejam fazer parte daquilo que movimenta todo o mundo Agronomia é a área que reúne técnicas para aprimorar produtos agrícolas e pecuários. Os agrônomos são responsáveis pelo cultivo dos solos e participam de todo o processo de produção agrícola, desde o preparo das sementes até a distribuição da safra para o consumidor final. Pecuária é a criação de animais para o consumo humano e para fins comerciais.
